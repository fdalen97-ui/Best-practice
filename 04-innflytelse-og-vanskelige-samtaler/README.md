# 04 — Innflytelse og vanskelige samtaler

## Hva er dette

Si nei uten relasjonsskade, korrigerende feedback ansikt-til-ansikt, lede samtaler jeg helst ville unngått, forhandle scope/mandat, motstå påvirkning rettet mot meg.

## Hvorfor for meg spesifikt

Diplomatisk default — navigerer rundt friksjon, ikke gjennom. Dette er det mest direkte gapet mot leder-nivå. Senior Advisor-rollen krever å levere ubehag uten å miste relasjonen.

## Suksesskriterier 12 mnd

- ≥12 "vanskelige samtaler" tatt og logget (én per måned).
- ≥12 "kalibrert no" dokumentert (én per måned).
- Minst ett tilfelle av korrigerende feedback ansikt-til-ansikt til likemann eller overordnet.
