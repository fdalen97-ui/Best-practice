# Progresjon — kvartalsvis

Append-only. Én ~500-ords refleksjon per kvartal. Skriv rett inn — ikke rediger tidligere kvartaler.

Struktur per kvartal:

```
## Q? YYYY

**Primærområde:**
**Sekundærområde:**

**Hva flyttet seg i baseline?**
(Hvilke delferdigheter gikk opp, hvilke stod stille, hvilke gikk ned. Én setning per.)

**Viktigste refleksjon dette kvartalet:**
(Et tema, ikke en liste. Hva begynte du å se som du ikke så før?)

**Hva tok jeg feil om?**
(En konkret antakelse som ikke holdt. Bias-navn om mulig.)

**Én handling jeg ikke tok som jeg burde:**
(Den du utsatte. Hvorfor? Er den fortsatt aktuell?)

**Kobling mot 2026–2030-mål:**
(Nærmere/lenger fra? Hvilket?)

**Primær/sekundær neste kvartal:**
(Med én linje begrunnelse hver.)

**LinkedIn-kandidat:**
(Ja/nei + hvilken innsikt, eller "ingen" + hvorfor.)
```
