# 05 — Skriftlig klarhet for beslutningstakere

## Hva er dette

Håndverket: BLUF, Amazon 6-pager, PR/FAQ, Situation-Complication-Resolution, beslutningsnotat på én A4.

## Hvorfor for meg spesifikt

Skriver godt, men langt og resonnerende. På 900k+-nivå er valutaen én side direktøren signerer uten oppfølgingsspørsmål. Ikke "skrive mer" — skrive strammere.

## Suksesskriterier 12 mnd

- ≥12 A4-beslutningsnotater skrevet som øvelse (ukentlig/månedlig).
- Minst 3 notater revidert fra langformat til 250 ord, med bevisst refleksjon på hva som mistet/holdt.
- Minst én Thomas-review som lander "ville signert uten oppfølgingsspørsmål".
