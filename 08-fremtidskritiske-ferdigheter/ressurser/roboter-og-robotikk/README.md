# Roboter og robotikk

Delkatalog for notater om embodied AI, VLA-modeller (Vision-Language-Action), humanoider og arbeidsrelevant robotikk.

## Hvorfor denne ligger her

Robotikk er ikke kjerneområdet mitt, men er den raskest bevegende frontieren i fysisk AI. Målet er å holde et **informert**, ikke hype-drevet syn, og kunne svare begrunnet når noen spør "hva betyr dette for eiendom/bygg/drift?".

## Kadens

Ett notat per kvartal. Se `praksis.md` i området for valgene (papir / demo / koblingsnotat).

## Filnavn

- `papir-YYYY-MM-<kortnavn>.md` — artikkel-gjennomgang
- `demo-YYYY-MM-<system>.md` — demo-analyse (hva er vist, hva er kuttet)
- `kobling-YYYY-QN-<tema>.md` — hvordan berører dette PropTech / bygg / drift
- `<forfatter>-<tittel>.md` — bok- eller YouTube-notater

## Begreps-rydding (legges til etter hvert)

Rask referanse for akronymer jeg ser ofte. Oppdateres når nye dukker opp.

- **VLA** — Vision-Language-Action model. Tar inn bilde + tekstinstruksjon, gir ut bevegelser.
- **VLM** — Vision-Language Model (uten action-head).
- **Embodied AI** — AI med fysisk kropp i en fysisk verden. Kontrast: chat-AI.
- **Teleop** — teleoperation. Mennesker styrer roboten for å samle treningsdata.
- **Sim-to-real** — overføring fra simulering til virkelig robot.
- **Dexterous manipulation** — finmotorikk, gripe-oppgaver.
- **Foundation model for robotics** — én modell, mange oppgaver/kropper (f.eks. π0, RT-2, Gemini Robotics).
