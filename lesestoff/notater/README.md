# Notater

Bok-oppsummeringer. Én fil per bok: `<forfatter>-<tittel>.md` (lowercase, bindestrek).

Mal: `.github/ISSUE_TEMPLATE/bok-oppsummering.md`. Samme struktur her.

Duplisering med områdespesifikke `ressurser/`? Valg per bok:
- **Her** når boka er generell eller spenner flere områder.
- **I område-ressurser** når den er tett koblet til ett spesifikt område.

Ikke lagre på begge steder. Symlink eller referanse.
