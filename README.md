# Kompetanseplattform

Personlig, strukturert utviklingsplattform for soft skills og fremtidskritiske ferdigheter. Horisont 2026–2030.

Ikke et lesebibliotek. Et arbeidsrom der lesing blir handling, koblet til konkrete jobbsituasjoner (PFT, RFP Studio, foredrag, workshops), med sporbar progresjon.

## Struktur

- `SPEC.md` — kildespesifikasjon. Ikke rediger uten grunn; redesign skjer bevisst.
- `00-oversikt/` — baseline, mål, kvartalsrytme, dashboard.
- `01-…` til `10-…` — ti kjerneområder, identisk intern struktur.
- `lederdagbok/` — ukentlig/månedlig råstoff til selvevaluering.
- `lesestoff/` — bibliotek, kø, bok-notater.
- `scripts/` — prompts/oppskrifter til ukes- og kvartalsrytme.

## Rytme

- **Ukentlig**: én refleksjonslogg i aktivt område.
- **Månedlig**: én boknotat, ett beslutningsnotat (øvelse), én vanskelig samtale, én "ikke-liste".
- **Kvartalsvis**: oppdater baseline + dashboard, skriv 500-ords refleksjon, velg primær/sekundær-område.
- **Årlig**: full gjennomgang + medarbeidersamtale-kobling.

## Prinsipper

Praksis over pensum. Baseline først. Refleksjon commites. Få nok, ikke for mye. Append-only. Privat først.

Se `SPEC.md` for full bakgrunn, områdebeskrivelser, suksesskriterier og anti-mønstre.
